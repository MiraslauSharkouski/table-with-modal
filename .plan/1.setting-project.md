# План разработки: Таблица с модальным окном (Vite + React 19 + Compiler + TS + AntD)

> **Стек**: Vite 6, React 19+, React Compiler (Forget), TypeScript 5.5+, Ant Design 5.x  
> **Задача**: CRUD-таблица с типизированной сортировкой, глобальным поиском и модальным окном
