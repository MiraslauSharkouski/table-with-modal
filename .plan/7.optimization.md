## 7. Оптимизации с React Compiler и AntD 5

- [ ] **Что делает React Compiler за нас**
  - [ ] Автоматическая мемоизация пропсов и значений между рендерами
  - [ ] Оптимизация зависимостей в `useEffect`/`useMemo`
  - [ ] Устранение лишних ререндеров при стабильных ссылках

- [ ] **Где всё ещё нужен ручной контроль**
  - [ ] `React.memo` для `TableActions`, если он не зависит от внешних пропсов
  - [ ] `useCallback` для обработчиков, передаваемых в `createColumns` (чтобы columns не пересоздавались)
  - [ ] Вынос `createColumns` вне компонента или мемоизация через `useMemo` с минимальными зависимостями

- [ ] **AntD 5.x особенности**
  - [ ] Использовать `ConfigProvider` для кастомизации темы (CSS-in-JS, no global styles)
  - [ ] Применять `theme={{ token: { borderRadius: 6 } }}` для согласованного дизайна
  - [ ] Иконки из `@ant-design/icons` — tree-shakable, импортировать только нужные

- [ ] **Производительность Vite**
  - [ ] Включить `build.target: 'esnext'` для современных браузеров
  - [ ] Использовать `manualChunks` для выноса `antd` и `dayjs` в отдельный бандл
  - [ ] Включить `sourcemap` в dev для удобной отладки
