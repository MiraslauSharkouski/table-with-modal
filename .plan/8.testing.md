## 8. Тестирование (Vitest + React Testing Library)

- [ ] **Настройка тестового окружения**
  - [ ] Установить: `npm install -D vitest @testing-library/react @testing-library/jest-dom jsdom`
  - [ ] Настроить `vitest.config.ts` с алиасами и окружением `jsdom`
  - [ ] Добавить скрипт: `"test": "vitest"`

- [ ] **Покрытие критических сценариев**
  - [ ] Рендер таблицы с моковыми данными
  - [ ] Добавление записи: открыть модалку → заполнить → сабмит → проверка `dataSource`
  - [ ] Редактирование: кликнуть "редактировать" → изменить поле → проверить обновление
  - [ ] Удаление: подтвердить → проверить исчезновение строки

- [ ] **Тесты сортировки и поиска**
  - [ ] Сортировка по имени: проверка порядка `localeCompare('ru')`
  - [ ] Поиск: частичное совпадение, регистронезависимость, очистка
  - [ ] Комбинация: поиск + сортировка → корректный результат

- [ ] **a11y-проверки**
  - [ ] Запуск `axe-core` через `@testing-library/react` или Playwright
  - [ ] Проверка `aria-label` у кнопок действий, фокус-ловушка в модалке
