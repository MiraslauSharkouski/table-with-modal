## 6. Модальное окно формы (AntD Form + React 19)

- [ ] **Структура модального окна**
  - [ ] Использовать `Modal` + `Form` из AntD
  - [ ] Props: `open: boolean`, `title: string`, `initialValues?: EntityFormValues`, `onSubmit`, `onCancel`
  - [ ] Использовать `Form.useForm()` внутри для контроля валидации

- [ ] **Поля формы с валидацией**
  - [ ] `name`: `<Input />`, правила: `{ required: true, min: 2, max: 100 }`
  - [ ] `date`: `<DatePicker format="DD.MM.YYYY" />`, правило `required`, парсинг в ISO
  - [ ] `value`: `<InputNumber min={0} step={0.01} />`, правило `required`
  - [ ] Сообщения об ошибках: `message: 'Обязательное поле'`, `validator` для кастомных проверок

- [ ] **Режимы: Create vs Edit**
  - [ ] При открытии на создание: `form.resetFields()`, `editingId = null`
  - [ ] При открытии на редактирование: `form.setFieldsValue(initialValues)`
  - [ ] При сабмите:
    ```ts
    try {
      const values = await form.validateFields();
      const payload = {
        ...values,
        date: dayjs(values.date).toISOString(), // нормализация
      };
      onSubmit(payload); // иммутабельное обновление в родителе
      form.resetFields();
      onCancel();
    } catch (err) {
      // AntD Form сам покажет ошибки
    }
    ```

- [ ] **Улучшения UX**
  - [ ] Закрытие по `Esc` и клику вне (AntD по умолчанию)
  - [ ] Блокировка кнопки "Сохранить" при `form.isFieldsTouched() && !form.isValidating`
  - [ ] Подтверждение удаления: `Modal.confirm({ title: 'Удалить?', onOk: () => onDelete(id) })`
