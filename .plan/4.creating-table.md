## 4. Реализация таблицы на AntD 6.x

- [ ] **Базовая настройка Table**
  - [ ] Передать `dataSource`, `columns`, `rowKey="id"`
  - [ ] Настроить `pagination={false}` (или включить при росте данных)
  - [ ] Добавить `loading`-пропс для будущих асинхронных сценариев
  - [ ] Использовать `ConfigProvider` для глобальной темы (опционально)

- [ ] **Определение колонок (вынести вне компонента для стабильности)**

  ```ts
  // features/entity-table/components/EntityTable/columns.tsx
  export const createColumns = (
    onEdit: (id: string) => void,
    onDelete: (id: string) => void
  ): ColumnType<EntityItem>[] => [
    {
      title: 'Имя',
      dataIndex: 'name',
      key: 'name',
      sorter: (a, b) => a.name.localeCompare(b.name, 'ru'),
      sortDirections: ['ascend', 'descend'],
    },
    {
      title: 'Дата',
      dataIndex: 'date',
      key: 'date',
      render: (date: string) => dayjs(date).format('DD.MM.YYYY'),
      sorter: (a, b) => new Date(a.date).getTime() - new Date(b.date).getTime(),
    },
    {
      title: 'Значение',
      dataIndex: 'value',
      key: 'value',
      render: (val: number) => new Intl.NumberFormat('ru-RU').format(val),
      sorter: (a, b) => a.value - b.value,
      align: 'right',
    },
    {
      title: 'Действия',
      key: 'actions',
      render: (_, record) => (
        <TableActions
          onEdit={() => onEdit(record.id)}
          onDelete={() => onDelete(record.id)}
        />
      ),
      width: 100,
    },
  ];
  ```

- [ ] **Типизированная сортировка (утилита)**
  - [ ] Создать `getTypedSorter<K extends keyof EntityItem>(field: K)`:
    ```ts
    export const getTypedSorter = <K extends keyof EntityItem>(field: K) => {
      const sorters: Record<K, CompareFn<EntityItem>> = {
        name: (a, b) => a.name.localeCompare(b.name, "ru"),
        date: (a, b) => new Date(a.date).getTime() - new Date(b.date).getTime(),
        value: (a, b) => a.value - b.value,
        id: () => 0, // не сортируем по ID
      };
      return sorters[field];
    };
    ```
  - [ ] Привязать к колонкам: `sorter: getTypedSorter('name')`
